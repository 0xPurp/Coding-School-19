Notes - GIT :


Tuto git :



git clone URL = Récupère un repository à partir d'une adresse URL

git add --all = Commit tout les fichiers du dossier dans le quel vous êtes

git commit -m"­des­cri­pti­on" = Ajout d'un commentaire à tout les fichiers modifié

git status = Affiche l'état du repository

git tag étiqu­ette = Pose une étiqu­ette de version sur les fichiers

git push origin master = Envoie les modifi­cations sur le repository principal

git log = Affiche l'hist­orique des commandes (q pour sortir)

git config --list = Affiche la liste des paramètres de git pratique pour check si vous avez bien rentré vos iddentifiants

git config --global user.name NomUt­ili­sateur && git config --global user.email email = Enregistre votre nom d'utilisateur ainsi que votre email
