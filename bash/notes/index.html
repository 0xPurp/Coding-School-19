Notes - Bash :


Liste des commandes:

  • pwd : Montre le chemin absolu du repertoire.
  • cd : Change de fichier.
  • ls : Afficher ce qu'il y a dans le fichier.
  • touch : Créer un fichier.
  • rm : Supprime un fichier.
    • rm -rf : Suprimme un dossier rempli.

  • mkdir : Crée un dossier.
  • rmdir : Supprimer un dossier vide.
  • mv : Renommer un fichier/dossier ou le déplacer.
  • cp : Copier un fichier/dossier.
  • tree : Montre l'arborescence de nos fichiers.
  • man : Ouvre la documentation de la commande (q pour quitter)
